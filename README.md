
# Decision Quality Lab

An open-source AI-assisted decision quality tool that helps leaders improve important decisions through structured reasoning, risk assessment, governance, and human judgment.

## Vision

Better decisions through better thinking.

## Features

- Decision Canvas
- Assumption Review
- Risk Assessment
- Alternative Analysis
- Governance Checklist
- Executive Decision Summary

## Roadmap

- Interactive web interface
- AI-assisted recommendations
- Decision scoring
- PDF export
- Decision history

## Tech Stack

- HTML
- CSS
- JavaScript
